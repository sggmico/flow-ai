# Code Review Guide

为评审、建立标准或辅导提供可执行的评审方法与输出结构。

详细方法论、模板与清单见 `README.md`。

## 执行方式

1. 明确目标与范围（PR 评审 / 团队标准 / 辅导）。
2. 输出精炼评审流程：上下文了解 -> 高层审视 -> 逐行检查 -> 结论与下一步。
3. 给出可执行清单（正确性 / 安全 / 性能 / 测试 / 可维护性）。
4. 输出反馈语言规范：具体、可操作、标注严重性。
5. 需要时提供评审模板与建议节奏（时限、PR 大小）。

## 输出要求

- 结构化、可执行、可落地。
- 不输出无关技术细节或超出范围的实现建议。
